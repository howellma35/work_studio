# 车辆操作手册摘录

## 胎压监测系统 (TPMS)
- 查看胎压：中控屏 → 设置 → 服务 → 胎压监测
- 标准胎压值（冷胎状态）：
  - 前轮：2.9 bar (42 psi)
  - 后轮：3.0 bar (43 psi)
- 胎压过低警告：低于 2.5 bar 时屏幕显示黄色警告
- 胎压过高警告：超过 3.5 bar 时屏幕显示红色警告
- 重置胎压监测：充气后长按重置按钮3秒

## 充电相关
### 家用充电桩（7kW标准充电）
- 充电时间：约8-10小时从20%充至90%
- 建议：每晚充电，保持电量在20%-90%之间以延长电池寿命
- 充电口：左侧后方

### 超级充电站（120kW V2 / 250kW V3）
- V2超充：约45分钟从10%充至80%
- V3超充：约25分钟从10%充至80%
- 超充费用：1.8元/kWh（2026年7月价格）
- 预热电池：导航至超充站时自动预热（提升充电速度）

### 充电注意事项
- 不要经常充至100%（除长途出行外）
- 长期停放保持电量50%左右
- 低温环境充电速度会降低

## 蓝牙连接
- 连接步骤：中控屏 → 设置 → 蓝牙 → 配对新设备
- 最多同时连接2个蓝牙设备（1手机+1耳机）
- 支持蓝牙电话和蓝牙音乐
- 已配对设备自动连接

## 前备箱 (Frunk)
- 打开方式1：中控屏点击前备箱按钮（双击）
- 打开方式2：方向盘下方拉环
- 容量：约67升
- 用途：充电线缆、雨伞、小型行李

## 紧急开门
- 车内紧急开门：拉动车门下方机械释放拉环（位于车窗开关面板下方）
- 车外紧急开门：前门有机械钥匙孔（隐藏式），后门需从内侧操作
- 注意：12V小电池失效时电子门锁无法工作，需使用机械方式

## 常见故障处理

### 胎压报警
1. 减速至安全速度（低于60km/h）
2. 检查轮胎是否有明显损伤
3. 如扎钉：不要拔出钉子，低速行驶至最近修理店
4. 如爆胎：安全停车，联系特斯拉道路救援（400-910-0707）

### 电池电量过低
1. 立即关闭空调和座椅加热等耗电功能
2. 切换至"省电模式"（设置 → 驾驶 → 省电模式）
3. 寻找最近充电站（导航会自动推荐）
4. 低于5%时联系道路救援

### 12V小电池故障
- 症状：中控屏无法启动、车门电子锁失效
- 处理：使用机械方式开门，联系服务中心更换12V电池
- 12V电池寿命：约2-3年

### 系统重启
- 普通重启：中控屏 → 设置 → 安全 → 重启
- 硬重启：同时按住方向盘两侧滚轮按钮10秒
- 注意：重启不影响驾驶功能（转向、刹车、加速正常）
