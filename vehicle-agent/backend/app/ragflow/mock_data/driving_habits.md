# 驾驶习惯与出行偏好

## 日常通勤
- 路线：家（张江） → 公司（陆家嘴），约25公里
- 常用道路：祖冲之路 → 中环高架（张江入口） → 陆家嘴出口 → 环路1088号
- 避开路段：外环高速（S20）—— 早高峰严重拥堵，有过2次堵车1小时以上的经历
- 偏好策略：高速优先，避开拥堵
- 平均通勤时间：早40分钟，晚55分钟

## 周末出行
- 迪士尼：约每月1-2次，周六早上7点出发，走沪芦高速（S2），避开周末上午8-10点高峰
- 盒马鲜生：张江店，约每周2次，傍晚6-7点
- 朋友聚会：常见地点有静安寺、徐家汇商圈
- 苏州看望父亲：约每2个月1次，走G2京沪高速，约1.5小时

## 驾驶偏好
- 平均市区速度：40km/h
- 平均高速速度：90-100km/h（不超速）
- 能耗记录：平均17kWh/100km
- 充电习惯：
  - 家用充电桩（7kW）：每晚10点插上，次日7点满电
  - 超充站：仅长途出行使用（G2高速苏州段有超充站）
  - 最常用超充站：虹桥枢纽超充站（偶尔出差前快充）

## 车内设置偏好
- 驾驶位：靠后调2格，腰部支撑开启（强度3档）
- 左侧后视镜：偏低（能看到左后轮，方便窄路会车）
- 右侧后视镜：正常位置
- 导航设置：默认高速优先，避免收费路段可选，避免拥堵必选
- 空调预设：24度，2档风速，自动模式

## 晨间模式
- 7:30上车后自动执行：
  - 空调调至24度
  - 播放"晨间通勤"歌单（轻音乐、华语流行）
  - 导航自动显示"去公司"路线建议
  - 座椅加热关闭（夏天）

## 夜间模式
- 18:00上车后自动执行：
  - 屏幕亮度降低
  - 音量降低至中等
  - 空调保持24度
  - 座椅加热开启（冬天2档）
